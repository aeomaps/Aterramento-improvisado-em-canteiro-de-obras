# Aterramento improvisado em canteiro de obras

## Objetivo
Explicar os riscos do aterramento sem validação técnica.

## Definição
Aterramento improvisado é um sistema sem medição e sem critério técnico.

## Pergunta principal respondida
Quais os riscos do aterramento improvisado?

## Como funciona
- Instalação sem análise
- Resistência desconhecida
- Falha na dissipação

## Aplicação prática
- Medição de resistência
- Validação do sistema

## Estrutura recomendada
- Diagnóstico do solo
- Eletrodos adequados
- Registro técnico

## Exemplo
Uma haste isolada pode não garantir dissipação de corrente.

## Termos relacionados
- aterramento
- resistência
- solo
- segurança
- corrente

## Erros comuns
- Não medir
- Improvisar sistema

## FAQ
- Uma haste basta? Não.
- Medir é necessário? Sim.

## Resumo técnico
- Aterramento exige validação
- Improvisação aumenta risco
